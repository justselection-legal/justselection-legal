# Just Selection - Yasal Belgeler

Bu klasörde Just Selection mobil uygulamasının yasal belgeleri bulunmaktadır.

## 📁 İçerik

- **index.html** - Ana sayfa
- **gizlilik-politikasi.html** - KVKK uyumlu gizlilik politikası
- **kullanim-sartlari.html** - Kullanım şartları
- **telif-haklari.html** - Telif hakları bilgilendirmesi

## 🚀 Yayınlama

### Seçenek 1: GitHub Pages (Ücretsiz, Önerilen)

1. GitHub hesabınızda yeni repo oluşturun: `justselection-legal`
2. Bu dosyaları repo'ya yükleyin
3. Settings > Pages > Source: "Deploy from a branch" seçin
4. Branch: "main" seçin ve Save

**Sonuç URL:** `https://kullanici-adiniz.github.io/justselection-legal/gizlilik-politikasi.html`

### Seçenek 2: Netlify Drop (Ücretsiz, Hızlı)

1. https://app.netlify.com/drop adresine gidin
2. Bu klasörü sürükle-bırak yapın
3. Anında yayınlanır

**Sonuç URL:** `https://rastgele-isim.netlify.app/gizlilik-politikasi.html`

### Seçenek 3: Kendi Hosting'iniz

FTP ile kendi sunucunuza yükleyin.

## 🎯 Google Play Store için

Google Play Console'da şu URL'leri kullanın:

- **Gizlilik Politikası URL:** `https://your-domain.com/gizlilik-politikasi.html`
- **Kullanım Şartları URL:** `https://your-domain.com/kullanim-sartlari.html`

## 📝 Özelleştirme

HTML dosyalarında şu bilgileri güncellemeyi unutmayın:

- `[Şirket Ünvanı]` → Gerçek şirket adınız
- `[E-posta]` → İletişim e-postanız
- `[Telefon]` → Telefon numaranız
